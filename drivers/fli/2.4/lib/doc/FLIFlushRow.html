int FLIFlushRow

(flicam_t cam, int rows, int repeat)

Flush rows of a given virtual camera.

Documentation

Flush rows of a given virtual camera. This function flushes rows rows of virtual camera cam, repeat times.

Parameters:
cam - Virtual camera to flush rows of.
rows - Number of rows to flush.
repeat - Number of times to flush each row.
Returns:
Zero on success.
Non-zero on failure.
