int FLIGetFirmwareRev

(flidev_t dev, int* firmrev)

Get the firmware revision number of a given physical camera device.

Documentation

Get the firmware revision number of a given physical camera device. This function copies the firmware revision number for physical device dev into the location pointed to by firmrev.

Parameters:
dev - Physical camera device to find firmware revision number of.
firmrev - Pointer to a integer location where the firmware revision number is to be placed.
Returns:
Zero on success.
Non-zero on failure.
