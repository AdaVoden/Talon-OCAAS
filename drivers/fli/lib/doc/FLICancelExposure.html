int FLICancelExposure(flicam_t cam)

Cancel an exposure.

Documentation

Cancel an exposure. This function cancels an exposure in progress by closing the shutter.

Parameters:
cam - Virtual camera to cancel exposure of.
Returns:
Zero on success.
Non-zero on failure.
