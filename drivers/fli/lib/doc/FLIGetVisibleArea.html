int FLIGetVisibleArea(flicam_t cam, int *ul_x, int *ul_y, int *lr_x, int *lr_y)

Get the visible area of the given virtual camera.

Documentation

Get the visible area of the given virtual camera. This function finds the visible area of the CCD array for virtual camera cam. This area is specified in terms of a upper-left point and a lower-right point. The upper-left x-coordinate is placed in ul_x, the upper-left y-coordinate is placed in ul_y, the lower-right x-coordinate is placed in lr_x, the lower-right y-coordinate is placed in lr_y.

Parameters:
cam - Virtual camera to find visible area of.
ul_x - Pointer to where the upper-left x-coordinate is to be placed.
ul_y - Pointer to where the upper-left y-coordinate is to be placed.
lr_x - Pointer to where the lower-right x-coordinate is to be placed.
lr_y - Pointer to where the lower-right y-coordinate is to be placed.
Returns:
Zero on success.
Non-zero on failure.
