ATF Remote Observers Page

Welcome to The Automated Telescope Remote Observers Page. The Table of Contents is designed to assist you in completing you observing request.

Before you can observe, it is necessary to have a remote observer's code. This is a three letter code which allows us to tag your images for retrieval. It also gives us a nice idea of where everyone is from.

Once you have an observers code, you can go straight to the Observation Submission Form. The observer's guide will assist you in preparing your observing proposal.

You will also find links to current weather conditions and status of the telescope. It is also possible to see the most recently acquired image from the ATF.

Thanks again fro using the ATF. If you have any comments, please send them to atfproj@astro.physics.uiowa.edu.
